提权（这里有多种方法）

方法一（内核提权漏洞）

查询资料得知该版本存在内核提取漏洞：

CVE-2017-16995 (Ubuntu 16.04 BPF 提权)

操作步骤：

  • 在 Kali 上搜索该漏洞：

  • 你会找到一个名为 45010.c 的文件。
  • 传输文件： 在 Kali 上启动一个 Python HTTP 服务，在靶机上用 wget 下载它。
  • (靶机只允许443端口出站，试了80端口不行，必须先停止反弹shell)

然后恢复靶机shell

编译运行：

  • 运行后，直接就是 root 了。

方法二（SUID提权）

查找资料发现pkexec有一个漏洞

CVE-2021-4034

前往github寻找利用脚本

pwnkit/pwnkit.c at main · pyhrr0/pwnkit

下载下来通过蚁剑上传

根据README来编译使用

运行后拿到root权限
